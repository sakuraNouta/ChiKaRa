# idea快捷键
```
ctrl+shift+a 查找命令
alt+insert 新建，可在class或function上新建
ctrl+tab 切换
ctrl+n 搜索class名
ctrl+shift+n 搜索filename
ctrl+shitf+alt+n 搜索符号
shift + entry 新一行，可用于行尾跳出括号等
ctrl + 鼠标指定 显示提示
ctrl + q 显示javadoc 在 ctrl+q 放大
ctrl+s保存 ctrl+f9编译 热部署二连
ctrl+f10 触发update可以选择更新资源等
自定义 ctrl+shift+` distraction free mode
alt + shift + mouse click 选择多个地方编辑
alt+j 选择相同符号 shift+j反向
ctrl+shift+alt+j 全选
```
